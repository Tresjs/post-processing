# Pixelation

<DocsDemo>
  <PixelationDemo />
</DocsDemo>

Pixelation is an effect that pixelates the scene.

## Usage

```vue
<script setup lang="ts">
import { EffectComposer, Pixelation } from '@tresjs/post-processing'
</script>

<template>
  <EffectComposer>
    <Pixelation />
  </EffectComposer>
</template>
```

## Props

| Prop        | Description                    | Default |
| ----------- | ------------------------------ | ------- |
| granularity | The granularity of the pixels. | 30      |
